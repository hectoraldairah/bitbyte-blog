     25 ### Overview
     26 
     27 This website is primarly made with [Eleventy](https://www.11ty.dev/), a Static Site Generator (SSG) powered by Node.js. A good point of start is to check [this](https://www.zachleat.com/web/eleventy-tutorial-level-1/).

     29 You can explore all my source code [here](), if u want to fork it, u're more than welcome to.

     31 The goal is simple: to provide anyone interested in building a website with a fully transparent example they can read, edit, and understand — line by line.

     34 ### Styling
     35 
     36 I use a rather peculiar approach to styling my site. Instead of relying on a popular CSS framework like Tailwind, I work with SASS, paired with [Gorko](https://github.com/Andy-set-studio/gorko), a utility-first library that helps me create consistent design “units.”. Follwing the [CUBE CSS](https://cube.fyi/) methodology.
