     14 So, in order to get better at making pixel art I started to get into tutorials for learning.
     15 
     16 For now, my first learning source was Youtube. I started to watch a lot of pixelart art channels for begginer tips and tricks to start making my first compositions.
     17 
     18 One thing I learned is not to start making big pixelarts canvas, instead you can start with a 8x8 canvas. It's more easier to create very little drawings and you have more control about space and colors.
     19 
     20 This is my first composition, its like a Mario's Ghost but witk a pink ink. This drawing has very little variations (same color but different shades) to start making shadows and lights.
